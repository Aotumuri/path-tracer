座標タイムラインビューア

タイムスタンプ付きの座標 (x, y, z) を含む JSON ファイルを 1 つ以上選択してください。 各ファイルの最初の点を原点 (0, 0, 0) として扱います。

フォルダ選択は Chromium 系ブラウザ (Chrome / Edge / Arc など) での動作が安定しています。 Safari や Firefox ではフォルダ一括読み込みに対応していない場合があります。

追加レポート

エスケープレポートでは、CSV / JSON の軌跡を読み込み、指定した中心と半径からいつ脱出したかを一覧で確認できます。

STUCKレポートでは、x-z 平面の 変動を 750 step ウィンドウで解析し、標準偏差が閾値以下になった停滞区間を集計できます。

ズーム: 100%
